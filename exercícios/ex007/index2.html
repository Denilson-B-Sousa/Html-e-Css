As tags de marcação serve para definir algumas características, que possuem um texto como por exemplo: citações, Marcação, Texto superior, Texto inferior, Texto deletado.

Para marcar um texto usa-se a tag chamada <mark>., para ter a cor diferente utiliza-se css

Para colocar uma fórmula como está X2 Usa-se a tag <Sup>.

Para colocar uma fórmula química como está H2O usa-se a tag <sub>.

Para desconsiderar um texto usa-se a tag <del>.

Letras pequenas é conseguido com a tag <small>.

Para colocarnegrito no texto Usa-se a tag <strong>

Para colocar um texto em Itálico utiliza-se a tag <em>
